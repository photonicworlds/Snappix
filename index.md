# Snappix

Snappix is a classroom utility tool that picks students from Google Classroom courses and rosters in a simple, user‑friendly interface. It would replace a fair pick using a cup with popsicle sticks.

## What Snappix Does
- Allows users to sign in with their Google account
- Reads Google Classroom course lists
- Reads Google Classroom rosters
- Displays basic profile information (name, email) for clarity only (helpful if users have multiple accounts)
- If you love it, please donate (completely optional)

Snappix does **NOT** store, transmit, or share user data. All data stays on the user’s device. Made by teachers, for teachers.
All feedback is welcome but any ideas will become the sole property of PhotonicWorlds.
