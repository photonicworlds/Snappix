Snappix is utility for teachers that reads student names from a Google Classroom and picks from them at random (similar to a spinner). Built with simplicity in mind, Snappix focuses on speed of setup. The user can pick 1-5 students at random either with replacement or without. A school name and school logo can be customized.

---

## ✨ Features

- ⚡ **Google Classroom** — instantly download rosters from GC
- 🖼️ **Clean UI** — simple, distraction‑free interface (once per year setup)
- 🔧 **Configurable settings** — choose school name and logo
- 🧩 **Safe** — out-of-the-box network safety, no firewall rules needed
- 🔐 **Secure** — information encrypted to disk with AES256

---

# 🚀 Installing Snappix
- Download the latest Snappix ZIP file from the Releases section.
- Right‑click the ZIP and choose Extract All…
- Select a folder where you want Snappix to live (for example D:\Snappix) and extract.
- Open the folder and double‑click snappix.exe to launch the app.
- If you want quicker access, right‑click snappix.exe and choose:
  Send to → Desktop (create shortcut)
