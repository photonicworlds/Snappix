# Snappix
For teachers: Windows Unity program that reads student names from Google Classrooms per Google account. Then, the user can pick 1-5 students at random either with replacement or without. A school name and school logo can be customized.

# Installing Snappix
- Download the latest Snappix ZIP file from the Releases section.
- Right‑click the ZIP and choose Extract All…
- Select a folder where you want Snappix to live (for example D:\Snappix) and extract.
- Open the folder and double‑click snappix.exe to launch the app.
- If you want quicker access, right‑click snappix.exe and choose:
  Send to → Desktop (create shortcut)
